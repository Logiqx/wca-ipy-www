Senior Rankings

This project started on the SpeedSolving.com forum and provides unofficial rankings for the senior cubing community, produced from official WCA competition results.

  • Senior Rankings is list of senior competitors and their best results. It can be filtered by WCA event, age category and region
  • Recent Competitions is a list of recent competitions where senior competitors participated and links to their results
  • Future Competitions is a list of upcoming competitions with senior competitors registered and links to their WCA profiles

Useful information such as how to get your name added can be found in the Frequently Asked Questions (FAQ) and Privacy Notice.

Useful links

  • Senior Cubers Worldwide on Facebook is a great place to chat all things cubing
  • Older Cubers Discussions on SpeedSolving.com was where it all started and is still worth a visit
